# Style: Y2K Aesthetic (S18)

## Overview
- **ID**: S18
- **Name**: Y2K Aesthetic
- **Industries**: Lifestyle, Creative
- **Category**: Futuristic

## Design DNA
- **Palette Strategy**: Y2K aesthetic, glossy and futuristic 2000s style. Frosted and iridescent colors.
- **Typography**: Techno-style fonts (e.g., Michroma, Jura).
- **Radius Policy**: Large, rounded corners (16-32px).
- **Shadow Policy**: Soft, glossy reflections and iridescent glows.
- **Border Language**: Translucent borders with glass effects.
- **Patterns**: Bubbles, spheres, translucent blobs.
- **Motion**: Fluid, morphing animations.

## Rules
### Do
- Use iridescent and translucent colors
- Apply glossy reflection effects
- Use large rounded corners and blob shapes
- Incorporate Y2K-style techno fonts

### Don't
- Don't use flat colors exclusively
- Don't use sharp, industrial corners
- Don't use heavy, solid borders
- Don't use slow, rigid animations
